در این کارگاه، شرکت‌کنندگان با مفاهیم پایه سامانه‌های وب و سپس با **خودکارسازی سناریوهای تست** در ابزار **JMeter** آشنا می‌شوند.
::

## مراحل کلی کارگاه

::steps{level="3"}

### آشنایی با مفاهیم وب و JMeter
درک نحوه عملکرد پروتکل HTTP و معماری ابزار JMeter.

### پیاده‌سازی سناریوهای خودکار
طراحی، ضبط و توسعه سناریوهای تست برای سامانه‌های تحت وب.

### بهینه‌سازی و اشکال‌زدایی
افزایش کیفیت، پایداری و قابلیت نگهداشت سناریوهای تست.

::

## معرفی کارگاه

در این کارگاه ابتدا مفاهیم پایه مربوط به خودکارسازی سناریوهای سامانه‌های تحت وب شامل نحوه کارکرد درخواست‌های HTTP بررسی می‌شود.  
سپس شرکت‌کنندگان با نحوه **طراحی، ضبط، توسعه و اشکال‌زدایی سناریوهای خودکار** در ابزار **JMeter** آشنا می‌شوند.

این کارگاه پایه‌ای مناسب برای ورود به مباحث **تست کارایی و تست‌های پیشرفته‌تر** مبتنی بر JMeter فراهم می‌کند.

## سرفصل‌های کارگاه

::accordion

:::accordion-item{label="مفاهیم پایه"}
- مفاهیم ضروری مربوط به نحوه عملکرد سامانه‌های وب  
- بررسی پروتکل HTTP و نحوه تبادل درخواست و پاسخ  
- بررسی معماری و کلیات ابزار JMeter  
:::

:::accordion-item{label="مفاهیم پایه JMeter"}
- راه‌اندازی و نکات پیکربندی اولیه  
- ایجاد سناریوهای ساده در JMeter  
- بررسی اجزای اصلی مانند Thread Group، Sampler و Listener  
:::

:::accordion-item{label="روش‌های مختلف ضبط سناریو"}
- ضبط سناریو با Proxy در JMeter  
- هرس (Cleanup) سناریوهای ضبط‌شده  
- ضبط سناریو با ابزارهایی مانند BlazeMeter و Badboy  
:::

:::accordion-item{label="مدیریت کوکی‌ها در JMeter"}
- مفهوم Session و نقش Cookie در سناریوهای وب  
- استفاده از Cookie Manager و تنظیمات آن  
:::

:::accordion-item{label="انتقال اطلاعات بین درخواست‌ها"}
- وابسته‌سازی درخواست‌ها به یکدیگر  
- استفاده از متغیرها برای انتقال داده بین Samplerها  
:::

:::accordion-item{label="استخراج مقادیر مختلف از پاسخ‌ها"}
- استخراج داده از پاسخ‌های HTML و JSON  
- استفاده از Regular Expression Extractor و JSON Extractor  
:::

:::accordion-item{label="انواع Listenerها در JMeter"}
- بررسی Listenerهای پرکاربرد برای مشاهده نتایج  
- انتخاب Listener مناسب متناسب با هدف تست  
:::

:::accordion-item{label="کنترل‌کننده‌های روند خودکارسازی"}
- استفاده از Controllerها برای مدیریت جریان اجرا  
- پیاده‌سازی سناریوهای شرطی و تکرارشونده  
:::

:::accordion-item{label="Assertionها در JMeter"}
- بررسی انواع Assertionها  
- طراحی Assertionهای مؤثر برای اعتبارسنجی پاسخ  
:::

:::accordion-item{label="استفاده از Regular Expression"}
- مفاهیم پایه Regular Expression  
- کاربرد Regex در استخراج و اعتبارسنجی داده‌ها  
:::

:::accordion-item{label="روش‌های اتصال داده‌های آزمون"}
- استفاده از CSV Data Set Config  
- پیاده‌سازی سناریوهای Data-Driven  
:::

:::accordion-item{label="اشکال‌زدایی آزمون‌ها"}
- شناسایی خطاهای رایج در سناریوها  
- استفاده از ابزارهای Debug برای بررسی متغیرها و پاسخ‌ها  
:::

:::accordion-item{label="نکات تخصصی (SSL و افزونه‌ها)"}
- پیکربندی SSL برای سناریوهای امن  
- آشنایی با افزونه‌های کاربردی JMeter و موارد استفاده آن‌ها  
:::
